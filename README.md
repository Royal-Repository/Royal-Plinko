🎰 Premium Plinko Minigame for FiveM
პრემიუმ ხარისხის, სრულად ფუნქციონალური და ოპტიმიზირებული Plinko-ს მინი-თამაში თქვენი სერვერისთვის. შექმნილია საუკეთესო სათამაშო გამოცდილების უზრუნველსაყოფად.

✨ ძირითადი ფუნქციები (Features)
🔴 High-Density Board: 16-რიგიანი კლასიკური დაფა 17 სხვადასხვა მოსაგები სლოტით.

⚖️ Risk Levels: სამი სხვადასხვა რისკის დონე — Low, Medium და High შესაბამისი მამრავლებით (Multipliers).

💰 Multi-Ball Drop: ერთდროულად 20-მდე ბურთის ჩამოგდების შესაძლებლობა (Rain Effect).

🤖 Custom Dealer: სპეციალური NPC (Dealer), რომელთანაც ინტერაქციით იხსნება თამაში.

🎨 Modern UI: სუფთა, თანამედროვე და სწრაფი ინტერფეისი, რომელიც ადაპტირებულია ნებისმიერ რეზოლუციაზე.

🛡️ Server-Authoritative: თამაშის ლოგიკა და მოგება სრულად კონტროლდება სერვერულ მხარეს, რაც გამორიცხავს მანიპულაციას და ჩეთს.

🛠️ ტექნიკური დეტალები
Framework: სრული მხარდაჭერა Qbox / QBCore-ისთვის.

Target: იყენებს ox_target-ს მოთამაშესთან ინტერაქციისთვის.

Optimization: სკრიპტი არის მაქსიმალურად ოპტიმიზირებული (0.00ms idle-ზე).
